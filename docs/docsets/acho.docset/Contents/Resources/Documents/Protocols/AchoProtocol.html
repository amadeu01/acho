AchoProtocol

public protocol AchoProtocol

Protocol that defines an interface to ask the user to choose an option from a given list.

  • ask(question:options:)

    Prints the question and the options in the terminal and subscribes to key events to move the selection up and down. When the user presses enter, it returns the selected option.
